# تاي لي · Thai Lee — موقع المطعم

موقع تعريفي فاخر (Arabic RTL) لمطعم **تاي لي** التايلندي والآسيوي في جدة. مبني بـ HTML/CSS/JS خام (بدون أي build step) وجاهز للنشر على GitHub Pages.

## الرابط المباشر
بعد النشر: `https://az212z.github.io/batch9-restaurants/thai-lee/`
(يُحدَّث المسار حسب مكان الاستضافة النهائي.)

## المميزات
- **هوية «تايلندية جريئة»**: أحمر قرمزي + أسود فحمي + ذهبي دافئ + كريمي. خطوط El Messiri / Reem Kufi للعناوين و Tajawal للنصوص.
- **موشن توقيعي (Wok-Fire / Noodle)**: SVG ووك ثابت تتصاعد منه ألسنة لهب متعددة الطبقات (flicker عبر transform/opacity)، مع خيوط نودلز تُرسم وتتمايل (stroke-dashoffset + sway)، وحبيبات سمسم/بصل أخضر وبخار صاعد. مع **fallback كامل عند prefers-reduced-motion** (ووك ونودلز ثابتة، بدون نار متحركة).
- **بوليش إضافي**: ken-burns على صورة الطبق، scroll-reveal للأقسام، hover zoom على الكروت، تصغير الهيدر عند التمرير، وميض (sheen) على زر الـ CTA.
- **قائمة جوال ملء الشاشة** (100vw / 100dvh) بخلفية معتمة وزر X واضح.
- **إتاحة وصول**: تباين ≥ 4.5:1، focus-visible، aria-labels، HTML سيمانتيك، دعم لوحة المفاتيح، احترام reduced-motion.
- **نموذج حجز/طلب تجريبي**: يتحقق من المدخلات، يبني ملخّصًا، يحفظ نسخة في `localStorage`، ويعرض toast نجاح. موسوم بوضوح كـ **تجريبي**.
- **JSON-LD Restaurant** مع التقييم الحقيقي 4.3/1076 و servesCuisine: Thai/Asian، وOG/Twitter كاملة.

## التواصل / الـ Backend
هذا الفرع **لا يملك رقم هاتف/واتساب عام منشور** على خرائط قوقل، لذلك:
- أزرار الإجراء = **نموذج حجز/طلب تجريبي** (تحقّق + ملخّص + localStorage + toast) — للعرض فقط، لا يُرسل لأي خادم.
- **زر/FAB بارز لخرائط قوقل** للوصول إلى الموقع: `https://www.google.com/maps/search/?api=1&query=Thai%20lee%20Jeddah`
- قسم **«تابعنا»** للحسابات الاجتماعية.

> **عند التسليم**: يضيف صاحب المطعم رقم الواتساب/الهاتف، فنحوّل النموذج وأزرار CTA إلى روابط `wa.me/<رقم>` و `tel:` مباشرة. لم نخترع أي رقم.

